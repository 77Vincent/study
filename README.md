# Specs

* Runtime ----- nodejs@v.8.9.4
* Server ----- Express
* Client ----- React@v16.2.0
* Database ----- Mongodb
* CSS-precompiler ----- sass
* Javascript ----- ES6
* Build tool ----- Webpack

# Start 

client:
```sh
npm start

// port:3000
```

server: 
```sh
npm start

// port:3001
```
