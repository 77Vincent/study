# Server Documentation

## Development 
```bash
npm i
npm start
```

## Production
```bash
npm i
npm run server 
```

## Initialize database
* Initial data in ./scripts/init-db.js
```js
npm run db 
```

## Generate REST API Documentation
```bash
npm run doc
```

## Structure
* Configurations: ./config.js
* Entry file: ./index.js
* Functions: ./utils
* Routes: ./routes
* Models: ./models
* Static files: ./static
* Scripts for development: ./scripts
