## User Story
### 用户系统
* 可用于登陆的ID包括：手机号、邮箱
* 预留用户名，后端自动生成，客户端不暴露给用户，以便日后需要。
* 手机号、邮箱需有各自的验证
* 支持微信登陆
* 注册时不区分身份，默认皆为学生，后可申请成为老师，申请后通过认证身份才会变成老师。
* 老师可被搜索出来
