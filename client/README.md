# Frontend Instruction

## Specs 

+ react@16.0
+ redux@3.7.2
+ react-router-dom@4.2.2
+ webpack@3.10.0
+ fetch@2.0.3
+ less@2.7.1
+ antd@3.1.3

## To run 
```sh
npm i
npm run dev
```
//localhost:3000

## To build 
```sh
npm run build 
```

## UI
This project's UI is powered by Ant Design: https://ant.design/index-cn, custom theme is configured with this file: /theme.less.
